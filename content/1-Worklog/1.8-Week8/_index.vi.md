### Kết quả đạt được tuần 8:

* Hiểu được cơ chế quản lý danh tính và phân quyền trên AWS:
  * IAM User.
  * IAM Group.
  * IAM Policy.
  * IAM Role.
  * Nguyên tắc Least Privilege.

* Nắm được trách nhiệm bảo mật giữa AWS và người sử dụng thông qua Shared Responsibility Model.

* Tìm hiểu các dịch vụ hỗ trợ xác thực và bảo mật:
  * Amazon Cognito.
  * AWS Key Management Service (KMS).
  * AWS Security Hub.
  * AWS CloudTrail.

* Hoàn thành các bài thực hành bảo mật:
  * Tạo IAM Role cho EC2.
  * Cấu hình CloudTrail để theo dõi hoạt động.
  * Tạo và sử dụng KMS Key để mã hóa dữ liệu.
  * Kích hoạt Security Hub và xem các Security Findings.

* Tổng hợp kiến thức đã học trong chương trình AWS Bootcamp:
  * Cloud Fundamentals.
  * Networking.
  * VPC Connectivity.
  * Amazon EC2.
  * Amazon S3.
  * Security Services.

* Có khả năng áp dụng các dịch vụ bảo mật của AWS để xây dựng môi trường Cloud an toàn, đồng thời sẵn sàng triển khai vào dự án thực tế.
