### Week 3 Objectives:

* Learn how to design and manage network infrastructure using Amazon VPC.
* Understand AWS network security mechanisms and traffic control.
* Practice configuring core networking components for cloud deployments.

### Week 3 Achievements:

* Gained a solid understanding of how Amazon VPC provides isolated networking environments within AWS.

* Learned the roles of key networking components, including:
  * VPC
  * CIDR Block
  * Public Subnet
  * Private Subnet
  * Internet Gateway
  * NAT Gateway

* Understood the differences between Security Groups and Network ACLs, and how they are used to secure AWS network resources.

* Successfully completed hands-on networking exercises by:
  * Creating a custom VPC.
  * Configuring Public and Private Subnets.
  * Setting up Route Tables and an Internet Gateway.
  * Deploying a NAT Gateway for outbound Internet access from Private Subnets.
  * Testing network connectivity and validating the security configuration.

* Developed practical skills in building and managing AWS networking infrastructure, providing a strong foundation for deploying cloud applications in later stages of the training.
