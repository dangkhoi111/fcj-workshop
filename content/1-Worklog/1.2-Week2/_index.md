### Week 2 Achievements:

* Developed a solid understanding of Cloud Computing concepts and common cloud deployment models.

* Learned how AWS Global Infrastructure is organized and how Regions, Availability Zones, and Edge Locations improve service availability and scalability.

* Successfully created and configured an AWS Free Tier account.

* Applied essential security practices for AWS accounts, including:
  * Enabling Multi-Factor Authentication (MFA) for the Root User.
  * Creating IAM Users and IAM Groups.
  * Applying appropriate access control based on user responsibilities.

* Became familiar with AWS management tools such as:
  * AWS Management Console.
  * AWS CLI.
  * AWS SDK.

* Completed hands-on exercises related to AWS Budget management by:
  * Creating a Cost Budget.
  * Configuring a Usage Budget.
  * Setting up budget notifications for cost monitoring.

* Gained the ability to manage AWS accounts securely while monitoring cloud expenses, providing a strong foundation for future AWS infrastructure and deployment labs.
